Пользователь с паролем только в нижнем регистре может быть зарегистрирован
Шаги для воспроизведения:  
1. Перейдите на форму Registration;
2. Введите валидное значение в поле Username (например, Петя);
Low. Введите в поле Password значение 8-20 символов, в котором все символы в нижнем регистре (например, непрокатило);
Medium. Кликните кнопку ""Register"";

Ожидаемый результат: 
Под кнопками отображается валидационное сообщение: ""Please, provide valid data"". Юзер не зарегистрирован.

Фактический результат: 
Под кнопками отображается валидационное сообщение: ""Successfully registered! Please, click Back to return on login page"". Юзер зарегистрирован.
